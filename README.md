# Board App with Auth

## 실습 내용
- NestJS + TypeScript
- TypeORM, PostgreSQL 데이터베이스 연동
- NestJS Pipes
- JWT 인증, passpowrt.js
- 로그 log
- 설정 configuration

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
